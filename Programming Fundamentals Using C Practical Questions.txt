LIST OF PRACTICAL QUESTIONS:

1. Write a C program to print the sum and product of the digits of an integer number.

2. Write a C program to print the reverse of an integer number.

3. Write a C program to compute the sum of the first n terms of the series: S = 1-2+3-4+5...

4. Write a C program to check whether a given string is palindrome or not.

5. Write a C program to find whether a given number is prime or not. Use the same to generate the prime numbers less than 100.

6. Write a C program to display Fibonacci Series: 
	(i) Using Iteration, (ii) Using Recursion.

7. Write a C program to display Factorial Series: 
	(i) Using Iteration, (ii) Using Recursion.

8. Write a C program to calculate GCD of two numbers: 
	(i) Using Iteration, (ii) Using Recursion.

9. Write a C program that swaps two numbers using pointers.

10. Write a C program to perform the following operations on strings: 
	(i) String Concatenation, (ii) Compare two strings, (iii) Calculate the length of the strings.

11. Write a C program to create a structure Student containing fields for Roll No., Name, Class, Year and Total Marks. Create 10 students and store them in a file.
